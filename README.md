# Sky Sense - Weather App

A simple weather application that shows real-time weather information for any city using the OpenWeather API.

##  Live Demo
[https://skysense-weatherapp.vercel.app](https://skysense-weatherapp.vercel.app)

## Features
- Real-time weather data for any city
- Current temperature and weather conditions
- Humidity, wind speed, and more details
- Clean, mobile-friendly interface
- Fast loading times

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeather API
- Vercel

## Setup

1. Clone the repository:
```bash
git clone https://github.com/Nice-91/skysense_weatherapp.git
cd skysense_weatherapp
```

2. Get your free API key from [OpenWeather](https://openweathermap.org/api)

3. Add your API key to the JavaScript file:
```javascript
const API_KEY = 'your-api-key-here';
```

4. Open `index.html` in your browser
